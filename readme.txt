To view the sample, please install Liconcomp first, then unpack this folder and doubleclick the file with the extension "exe".
The Visual Basic .NET source code was created with Visual Studio 2008 and is available from the "source"-folder.
When the Program ist started, do this to show movies:
1. Press "Einschalten"
2. Press "Videos aus Verzeichnis laden"
3. Press "Alle Banden"
4. Click onto one of the filenames in the "Videosteuerung" selectbox.
5 videos should appear on the screen.

Um das Beispiel anzusehen, installieren Sie bitte zuerst Liconcomp, danach entpacken Sie diesen Ordner und doppelklicken auf die Datei mit der Endung "exe".
Der Visual Basic .NET Quellcode wurde mit Visual Studio 2008 erstellt und ist im "source"-Ordner verfügbar.
Nach dem Programmstart:
1. Drücken Sie auf "Einschalten"
2. Drücken Sie auf "Videos aus Verzeichnis laden"
3. Drücken Sie auf "Alle Banden"
